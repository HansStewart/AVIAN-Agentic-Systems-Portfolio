Automation 01 Active

Lead Pipeline Intelligence

An end-to-end agentic system that identifies, scores, and prepares outreach for AVIAN's highest-fit industrial accounts — fully automated, running on a 24-hour cycle.

How it was built
1
CRM data ingestion
HubSpot CRM contact and account records are ingested by n8n on a scheduled trigger. Each record includes company name, industry vertical, primary contact, and job title.
n8n Schedule Trigger HubSpot CRM node
2
ICP scoring
Each account is passed to a GPT-4o agent with a structured ICP scoring prompt. The agent evaluates industry fit, buyer role relevance, and strategic alignment, returning a score from 0–100.
OpenAI GPT-4o Structured prompt JSON output parsing
3
Tier classification
Accounts are classified into Hot (≥70), Warm (50–69), or Cold (<50) tiers based on their ICP score. Only hot accounts proceed to outreach generation.
n8n IF node Switch logic
4
Outreach generation
For each hot account, GPT-4o generates a personalised cold email, a LinkedIn connection message, and three call talking points — all role-aware and specific to the contact's title and company context.
OpenAI GPT-4o Role-aware prompting 3 output formats
5
CRM sync and alerting
All scored accounts are written back to HubSpot with their tier, score, and outreach copy attached to the contact record. Hot leads trigger an automated Gmail alert so nothing waits in a queue.
HubSpot CRM node Gmail API n8n Set node

How it was built
1
Content brief input
A content brief is submitted via a structured form — specifying the asset type (email, LinkedIn post, or blog intro), target audience, and key message focus.
n8n Form Trigger Structured input
2
AI content generation

OpenAI GPT-4o Brand voice prompt
3
Quality grading

Automation 03 Active

Re-Engagement System

A behavioral monitoring system that watches dormant prospects, flags contacts by inactivity window and original fit score, and generates priority-ranked follow-up sequences — so the highest-value opportunities never go cold undetected.

How it was built
1
Contact monitoring
n8n runs a daily check against HubSpot contact records, calculating the number of days since each contact's last touchpoint. The last activity date and original ICP score are both pulled into the workflow.
n8n Schedule Trigger HubSpot CRM node Date calculation
2
Dormancy threshold check
Contacts are filtered against two thresholds — 14 days and 30 days of inactivity. Only contacts exceeding at least one threshold are passed to the next stage.
n8n IF node 14-day / 30-day logic
3
Priority scoring
A GPT-4o agent evaluates each dormant contact using a composite priority score — weighing original ICP fit, days dormant, and last interaction type — and classifies each as Immediate, This Week, or This Quarter.
OpenAI GPT-4o Composite scoring 3-tier classification
4
Re-engagement copy generation
For each flagged contact, GPT-4o generates a contextual re-engagement message — referencing the previous interaction, acknowledging the gap naturally, and offering a relevant next step without pressure.
OpenAI GPT-4o Context-aware prompting
5
Priority queue and alerting
All flagged contacts are written back to HubSpot ranked by priority tier. Immediate-tier contacts trigger an instant Gmail alert to ensure same-day follow-up action.
HubSpot CRM node Gmail API Priority ranking
